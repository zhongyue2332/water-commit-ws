## [0.0.2]

**发布日期:** 2025-10-22

**新增功能：**
- 📋 提交前暂存区无内容自动执行 git add -A，无需手动暂存
- 🔄 支持自动推送到远程仓库，需手动启用:`waterCommit.autoSyncRemote=true`
- ⚠️ 推送失败时提供详细错误信息
- ⚙️ 自动暂存和自动推送支持用户配置
